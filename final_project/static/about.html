About Us

Website Grader is a grading web app which displays different web pages. Students and Instructors can login to the web app and view their individual dashboards. The students dashboard gives students the option to pick their top three favorite websites. The instructors dashboard has an upload and download option. Through the upload, the instructor upploads a csv file usernames, passwords and roles as well as a zip file of websites. The download option lets the instructor download the ranking of the various websites.

How does it work?

We've created this web app using Python Flask for the backend, mySQL to store information on the database and HTML/CSS/JS and React for the front-end.
